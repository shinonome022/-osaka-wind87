以下の手順に従って、サブエージェントを使用して。
全体の目的：大阪大学ウインドサーフィン部の紹介用ホームページの作成


ステップ１：リサーチエージェント
目的：大阪大学ウインドサーフィン部の紹介用ホームページの作成のための情報収集

サブエージェントを起動して、以下を実行
- 大阪大学ウィンドサーフィン部をリサーチ
- 紹介用の文章やサイトのアイディアを作成
- リサーチ結果と作成紹介文、アイディアをressearch.mdにまとめる。
- 完成したらgit add & git commit


ステップ２：コーダーエージェント
サブエージェントを起動して以下を実行：
- research.mdを参考にしてのindex.htmlを作成する
- CSS・JavaScriptも同じファイルに書く
- 完成したらgit add & git commit


ステップ３：レビュアーエージェント
別のサブエージェントを起動して以下を実行：
- ステップ１で作成したresearch.mdとステップ2で作成したindex.htmlを読み込む
- 問題点・改善点をreview.mdにまとめる
- 完成したらgit add & git commit

ステップ４：コーダーエージェント（修正）
別のサブエージェントを起動して以下を実行：
- review.mdの指摘を読み込む
- index.htmlを修正する
- 完成したらgit add & git commit
